# Опциональные конфиги

Профили TLP — **не обязательны**. Скрипт postinstall уже создаёт безопасный профиль `99-zorin-snappy.conf`. Эти файлы можно использовать, если нужен акцент на батарею или на производительность.

| Файл | Назначение |
|------|-------------|
| `tlp-battery.conf` | Максимум батареи: power/low-power на батарее, boost выключен. |
| `tlp-performance.conf` | Максимум производительности даже на батарее (быстрее разряд). |

**Установка вручную:**

```bash
# Из корня репо
sudo cp configs/tlp-battery.conf /etc/tlp.d/
sudo systemctl restart tlp
```

Порядок загрузки TLP: файлы в `/etc/tlp.d/` с большим номером переопределяют меньшие. Имена `99-zorin-*` имеют приоритет; при необходимости переименуй (например `98-tlp-battery.conf`), чтобы этот профиль имел приоритет над `99-zorin-snappy.conf`.
