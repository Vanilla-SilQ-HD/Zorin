# Опциональные конфиги

Профили TLP — **не обязательны**. Скрипт postinstall уже создаёт безопасный профиль `99-zorin-snappy.conf` («snappy»). Эти файлы можно использовать, если нужен более явный акцент на батарею, производительность или баланс.

| Файл | Назначение |
|------|-------------|
| `tlp-battery.conf` | Максимум батареи: режим power/low-power на батарее, boost выключен. |
| `tlp-balanced.conf` | Сбалансированный режим: compromise между батареей и отзывчивостью. |
| `tlp-performance.conf` | Максимум производительности даже на батарее (быстрее разряд). |

**Установка вручную:**

```bash
# Из корня репо (пример: профиль battery)
sudo cp configs/tlp-battery.conf /etc/tlp.d/
sudo systemctl restart tlp
```

Порядок загрузки TLP: файлы в `/etc/tlp.d/` с большим номером переопределяют меньшие. Имена `99-zorin-*` имеют приоритет; при необходимости переименуй (например `98-tlp-battery.conf`), чтобы этот профиль имел приоритет над `99-zorin-snappy.conf`.
